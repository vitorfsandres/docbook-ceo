# G025 — MVP mobile-first

**Decisão:** Priorizar o desenvolvimento na experiência em **smartphones (mobile-first)**, tratando a versão Web/desktop como secundária, pois o médico faz a maior parte dos inputs pelo celular no plantão. UX mobile em listas expansíveis (sanfona/accordion).

**Racional:** Reduzir carga cognitiva e cliques em ambiente de emergência; o uso real é majoritariamente no celular. Refina (não anula) a escolha por PWA de [[G012 - MVP como aplicação web responsiva (PWA) primeiro]].
