# G027 — Registrar ambas as marcas (Docbook + Evolumed) no INPI

**Decisão:** Registrar **imediatamente** as duas marcas — **"Docbook"** e **"Evolumed"** — no INPI (custo acessível ~R$ 800), para não perder nenhum nome e garantir segurança jurídica. "Docbook" vai com nome composto/descritivo (ex.: "Docbook — Sistema de Governança Médica"), com possível simplificação após 5–10 anos. Confirma a arquitetura de marca de [[G021 - Marca guarda-chuva Evolumed (Docbook vira módulo)]].

**Racional:** Registro é barato frente ao risco de perder os nomes; "Docbook" é genérico (prefixo "doc" saturado) e "Evolumed" é exclusivo/forte para o ecossistema.
