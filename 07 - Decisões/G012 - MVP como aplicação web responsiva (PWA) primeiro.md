# G012 — MVP como aplicação web responsiva (PWA) primeiro

**Decisão:** Focar o MVP em uma **aplicação web responsiva (PWA)**, deixando o app nativo para fase futura.

**Racional:** Agilidade e custo — front-end web estimado em R$ 15–20 mil vs. app mobile nativo em R$ 120–130 mil (estimativas de Emerson). Acelera o time-to-market.
